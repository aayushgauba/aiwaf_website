Getting Started

Welcome to AIWAF! This guide will help you get started with installing and configuring your AI-powered web application firewall.

Installation

Follow these steps to install AIWAF on your system:

# Download AIWAF curl -fsSL https://get.aiwaf.com/install.sh | bash # Start the service sudo systemctl start aiwaf sudo systemctl enable aiwaf

Basic Configuration

Configure AIWAF by editing the configuration file:

# Edit configuration sudo nano /etc/aiwaf/config.yml # Restart service after changes sudo systemctl restart aiwaf
Tip: Start with the default configuration and customize as needed for your specific environment.

Verification

Verify your installation is working correctly:

# Check service status sudo systemctl status aiwaf # View logs sudo journalctl -u aiwaf -f
