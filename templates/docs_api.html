API Reference

Complete reference for the AIWAF Management API. All endpoints require authentication using a bearer token.

Authentication

POST/api/v1/auth/token

Generate an authentication token for API access.

curl -X POST https://your-aiwaf:8080/api/v1/auth/token \ -H "Content-Type: application/json" \ -d '{"username": "admin", "password": "your-password"}'

Get Status

GET/api/v1/status

Retrieve current protection status and statistics.

curl -H "Authorization: Bearer YOUR_TOKEN" \ https://your-aiwaf:8080/api/v1/status Response: { "status": "active", "mode": "block", "threats_blocked": 1247, "requests_processed": 45829, "uptime": "15d 4h 32m" }

Update Configuration

PUT/api/v1/config

Update AIWAF configuration settings.

curl -X PUT https://your-aiwaf:8080/api/v1/config \ -H "Authorization: Bearer YOUR_TOKEN" \ -H "Content-Type: application/json" \ -d '{ "protection": { "mode": "block", "sensitivity": "high" } }'

Get Threat Log

GET/api/v1/threats

Retrieve recent threat detection events.

curl -H "Authorization: Bearer YOUR_TOKEN" \ "https://your-aiwaf:8080/api/v1/threats?limit=100&since=2024-01-01"
